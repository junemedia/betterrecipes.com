This plugin will automatically add Pinterest Pin buttons to all the images in your Wordpress content posts.

== Description ==
Once this plugin is activated it will automatically add the pinterest pin button to all the images within your content.

You will not have to change anything on your Wordpress site.

This will hook into the in-built Wordpress function to get the content, searches for all images inside the content and adds the pinterest button to the top right of the image.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload paulund-pinterest.php to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
